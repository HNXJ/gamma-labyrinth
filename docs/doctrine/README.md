# Gamma Labyrinth Doctrine Sources

This directory contains the generalized Control/Doctrine sources for Gamma Labyrinth.

These documents define scientific mission alignment, repo laws, project objectives, zero-tolerance rules, truth-safe discipline, and multi-agent coordination. They are not Truth-plane receipts and do not assert current active biological state, accepted circuit state, runtime state, or branch truth.

## Documents

1. `1_gamma_scientific_omission_incremental_growth_goals.md` — generalized omission/predictive-coding scientific mission and incremental N -> N+1 growth doctrine.
2. `2_gamma_repo_laws_tasks_debugging_improvement.md` — repo laws, task discipline, debugging, hygiene, evidence, and zero-tolerance rules.
3. `3_gamma_project_goals_rules_objectives.md` — project goals, non-negotiable laws, four-plane architecture, canonical repo roles, and agent orchestration.
